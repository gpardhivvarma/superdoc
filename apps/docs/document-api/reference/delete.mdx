## Summary

Delete content at a target position.

- Operation ID: `delete`
- API member path: `editor.doc.delete(...)`
- Mutates document: `yes`
- Idempotency: `conditional`
- Supports tracked mode: `yes`
- Supports dry run: `yes`
- Deterministic target resolution: `yes`

## Expected result

Returns a TextMutationReceipt with applied status; receipt reports NO_OP if the target range is already empty.

## Input fields

| Field | Type | Required | Description |
| --- | --- | --- | --- |
| `target` | TextAddress | yes | TextAddress |

### Example request

```json
{
  "target": {
    "blockId": "block-abc123",
    "kind": "text",
    "range": {
      "end": 10,
      "start": 0
    }
  }
}
```

## Output fields

_No fields._

## Pre-apply throws

- `TARGET_NOT_FOUND`
- `CAPABILITY_UNAVAILABLE`
- `INVALID_TARGET`

## Non-applied failure codes

- `NO_OP`
