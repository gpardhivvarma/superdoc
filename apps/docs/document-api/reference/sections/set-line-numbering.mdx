## Summary

Enable or configure line numbering for a section.

- Operation ID: `sections.setLineNumbering`
- API member path: `editor.doc.sections.setLineNumbering(...)`
- Mutates document: `yes`
- Idempotency: `conditional`
- Supports tracked mode: `no`
- Supports dry run: `yes`
- Deterministic target resolution: `yes`

## Expected result

Returns a SectionMutationResult receipt; reports NO_OP if line numbering settings already match.
