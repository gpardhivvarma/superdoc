## Summary

Set page-edge margins for a section.

- Operation ID: `sections.setPageMargins`
- API member path: `editor.doc.sections.setPageMargins(...)`
- Mutates document: `yes`
- Idempotency: `conditional`
- Supports tracked mode: `no`
