## Summary

Set vertical page alignment for a section.

- Operation ID: `sections.setVerticalAlign`
- API member path: `editor.doc.sections.setVerticalAlign(...)`
- Mutates document: `yes`
- Idempotency: `conditional`
- Supports tracked mode: `no`
- Supports dry run: `yes`
- Deterministic target resolution: `yes`

## Expected result

Returns a SectionMutationResult receipt; reports NO_OP if vertical alignment already matches.

## Input fields

| Field | Type | Required | Description |
| --- | --- | --- | --- |
| `target` | SectionAddress | yes | SectionAddress |
| `value` | enum | yes | `"top"`, `"center"`, `"bottom"`, `"both"` |

### Example request

```json
{
  "target": {

}
```
