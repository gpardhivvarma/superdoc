## Summary

Convert a text range into a table.

- Operation ID: `tables.convertFromText`
- API member path: `editor.doc.tables.convertFromText(...)`
- Mutates document: `yes`
- Idempotency: `non-idempotent`
- Supports tracked mode: `no`
- Supports dry run: `yes`
- Deterministic target resolution: `yes`

## Expected result

Returns a TableMutationResult receipt confirming text was converted into a table.
