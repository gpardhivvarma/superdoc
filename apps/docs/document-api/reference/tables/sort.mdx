## Summary

Sort table rows by a column value.

- Operation ID: `tables.sort`
- API member path: `editor.doc.tables.sort(...)`
- Mutates document: `yes`
- Idempotency: `non-idempotent`
- Supports tracked mode: `no`
- Supports dry run: `yes`
- Deterministic target resolution: `yes`

## Expected result

Returns a TableMutationResult receipt confirming rows were reordered.

## Input fields

| Field | Type | Required | Description |
| --- | --- | --- | --- |
| `keys` | object[] | yes |  |
| `nodeId` | string | no |  |
| `target` | BlockNodeAddress | no | BlockNodeAddress |

### Example request

```json
{
  "keys": [
    {
      "columnIndex": 1,
      "direction": "ascending",
      "type": "text"
    }
  ],
  "nodeId": "node-def456",
  "target": {
    "kind": "block",
    "nodeId": "node-def456",
    "nodeType": "paragraph"
  }
}
```

## Output fields

_No fields._

### Example response

```json
{
  "success": true,
  "table": {
    "kind": "block",

    }
  ]
}
```
