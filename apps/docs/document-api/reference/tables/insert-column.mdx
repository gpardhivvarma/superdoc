## Summary

Insert a new column into the target table.

- Operation ID: `tables.insertColumn`
- API member path: `editor.doc.tables.insertColumn(...)`
- Mutates document: `yes`
- Idempotency: `non-idempotent`
- Supports tracked mode: `yes`
