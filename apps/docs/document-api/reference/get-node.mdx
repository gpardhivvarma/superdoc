## Summary

Retrieve a single node by target position.

- Operation ID: `getNode`
- API member path: `editor.doc.getNode(...)`
- Mutates document: `no`
- Idempotency: `idempotent`
- Supports tracked mode: `no`
- Supports dry run: `no`
