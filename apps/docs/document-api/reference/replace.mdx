## Summary

Replace content at a target position with new text or inline content.

- Operation ID: `replace`
- API member path: `editor.doc.replace(...)`
- Mutates document: `yes`
- Idempotency: `conditional`
- Supports tracked mode: `yes`
- Supports dry run: `yes`
- Deterministic target resolution: `yes`

## Expected result

Returns a TextMutationReceipt with applied status; receipt reports NO_OP if the target range already contains identical content.
