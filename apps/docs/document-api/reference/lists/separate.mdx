## Summary

Split a list sequence at the target item, creating a new sequence from that point forward.

- Operation ID: `lists.separate`
- API member path: `editor.doc.lists.separate(...)`
- Mutates document: `yes`
- Idempotency: `conditional`
- Supports tracked mode: `no`
- Supports dry run: `yes`
- Deterministic target resolution: `yes`
